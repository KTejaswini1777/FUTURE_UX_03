# Elevate Spaces CRM Dashboard

## UI/UX Design Task 3 – Future Interns

Elevate Spaces CRM is a web-based client relationship management dashboard designed for a small interior design agency. The platform helps the team manage leads, active clients, project progress, tasks, and follow-ups in one centralized workspace.

## Project Overview

Small interior design agencies often manage enquiries, client details, site visits, project updates, and follow-ups across spreadsheets and messaging apps. This can make it difficult to track progress and may lead to missed follow-ups.

Elevate Spaces CRM was designed to create a clear workflow from a new lead to an active client and ongoing project management.

## Target Users

- Agency owner
- Interior designers
- Project coordinators
- Small agency team members

## Key Features

- Dashboard with business overview
- Lead tracking and visual sales pipeline
- Search and filter options for leads
- Client list management
- Client profile and engagement status
- Project progress tracking
- Activity timeline
- Task tracker with priorities and status
- Follow-up reminders

## CRM Workflow

New Lead → Contacted → Site Visit → Proposal → Won → Active Client → Project Progress → Follow-up

## Design Decisions

The dashboard uses a consistent sidebar navigation to make important CRM sections easy to access. Information is organized using cards, tables, status indicators, and pipeline stages so users can quickly understand their work.

A burgundy and blush color palette was used to maintain consistency with the existing Elevate Spaces brand identity.
